# Video Splitter & Background Replace

## Требования

**Обязательно должно быть установлено:**

- **Python 3**
- **FFmpeg** (должен быть добавлен в PATH)

---

Набор скриптов для автоматической нарезки длинных видео на короткие сегменты и наложения оверлея (замены фона).

Удобно для создания вертикальных видео (Reels, TikTok, Shorts) с анимированным оверлеем.

---

## Структура проекта
├── source/          ← исходные длинные видео (кладёте сюда)
├── result/          ← нарезанные сегменты (создаётся автоматически)
├── final/           ← готовые видео с оверлеем (создаётся автоматически)
├── overlay.mp4      ← файл с оверлеем
├── split.py         ← скрипт нарезки видео
├── bg_replace.py    ← скрипт наложения оверлея
└── .gitignore
text---

## Подготовка оверлея (overlay.mp4)

1. В CapCut создайте проект **1080×1920**.
2. Сделайте **ровный однотонный фон** (по умолчанию розовый `#FF3EAA`).
3. Разместите поверх фона все нужные элементы (текст, анимации, эффекты).
4. Экспортируйте видео.

> Цвет фона можно изменить в настройках скрипта.

---

## Как пользоваться

### Шаг 1: Нарезка видео

1. Поместите исходные видео в папку `source`.
2. Запустите скрипт:
   
   python split.py

Укажите:
Длину одного сегмента в секундах (например: 15, 30, 60)
y — с перекодированием (рекомендуется)
n — без перекодирования (быстрее)


Шаг 2: Наложение оверлея

Убедитесь, что файл overlay.mp4 находится в корне проекта.
Запустите скрипт: bg_replace.py

Готовые видео появятся в папке final.

Настройки (bg_replace.py)
PythonTARGET_W = 1080                    # ширина результата
TARGET_H = 1920                    # высота (9:16)

CHROMA_COLOR = "0xFF3EAA"          # цвет фона для вырезания
SIMILARITY = "0.09"                # чувствительность (0.05 — 0.25)
BLEND = "0.00"

OVERLAY_PAD_SECONDS = "0.5"        # запас в конце оверлея
Примеры цветов:

Зелёный: "0x00FF00"
Синий: "0x0000FF"
Чёрный: "0x000000"


Полезные советы

Перед большой партией протестируйте на 1–2 видео.
Если остаются цветные засветы — увеличьте SIMILARITY.
Для лучшего качества используйте режим y на этапе нарезки.
Готовые файлы имеют вид: video_part_001_final.mp4
